Pre-automation

  1. The current SI sessions are observed by two telescopes: Hobart12 and Hart15m. The default directory for the Hart15m SI data is flexbuf:/disk2/ht/.
  2. The Hobart12 data can be correlated directory off the flexbuff. To do so, run vbs_fs -I session* /mnt/vbsSI/ on flexbuffhb or flexbuf-2hb. The data can also be copied to another machine for correlation to avoid potential recording issues when another experiment is running.
  3. Other mounting or data directories may be used, but need to be added to the MySQL database. See General > Accessing the database.
  4. On a web browser, open http://dynob.tasphys.ga/web/si.html.
  5. Give the session name and press the start button. DO NOT close the browser when the process is running. It will end when the level1 vgosDB is created or when other errors occur.
  6. The correlation directory is at correlator:~/correlations/si/sessionname
