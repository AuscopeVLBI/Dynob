Pre-requisite

The Dynob feedback is done by translating the zero-km baseline source flux density (or median source flux density from observations for B-model sources) and the re-estimated antenna SEFD on the database into SKED-format flux and equip catalogues, respectively. Please make sure the Dynob performance monitoring step is completed first.

Creating the catalogue files:

  1. Open a web browser on ops8 and run http://dynob.tasphys.ga/scheduling/feedback.php?exp=aum048. Replace aum048 with the session's code. The "?exp=sessioncode" is optional. If absent, the latest source flux densities and antenna SEFDs ordered by session date (median from two latest sessions for the flux, and five for the SEFD) will be used.
  2. Dynob downloads the latest SKED flux and equip catalogues from github everytime.
  3. The new catalogue files will be created as dynflux.cat and dynequip.cat, stored under the scheduling/dynflux/ directory of Dynob. The download links will also be shown upon successful creation.
