Steps

  1. If the correlation was done using Dynob, run http://dynob.tasphys.ga/postcorr/initfringefit.php?exp=sessionname until a message pops up. Otherwise, do the following steps first.
  2. Create a folder (permission at least 775, user www-data) under Dynob root dir/tmp/sessionname
  3. Download the session's VEX file and store it at here.
  4. Check if the session index exists on the dynob > Correlation database. Create a new record if not.
  5. Then, you may run http://dynob.tasphys.ga/postcorr/initfringefit.php?exp=sessionname.
  6. After running initfringefit, run http://dynob.tasphys.ga/postcorr/fringefit.php?exp=sessionname until a message pops up.
